# @masqo/shared

Shared TypeScript types and constants for the Masqo ecosystem.

This package is an internal dependency used by `@masqo/engine` and `@masqo/cli`. You do not need to install it directly.

## Packages that use this

- [`@masqo/engine`](https://www.npmjs.com/package/@masqo/engine) - detection and redaction core
- [`@masqo/cli`](https://www.npmjs.com/package/@masqo/cli) - CLI and Claude Code hook
